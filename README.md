### Installation
```bash
$ [sudo] pip install values
```

#### Examples
```python
>>> import values
>>> values.get(1)
[1]

>>> values.get("string")
["string"]

>>> values.get([1,2])
[1,2]

>>> values.get(None)
[]
```
